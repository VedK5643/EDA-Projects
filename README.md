# EDA Projects
A collection of Exploratory Data Analysis notebooks on real-world datasets using Python.

## Projects
| Project | Dataset | Records | Notebook |
|---|---|---|---|
| IPL Data Analysis (2008–2024) | IPL Complete Dataset | 260,000+ deliveries | [View](./IPL-EDA/) |
| Superstore Sales Analysis | Sample Superstore | 9,994 | [View](./Superstore%20dataset%20EDA/) |
| Zomato Bangalore Analysis | Zomato Bangalore | 51,000+ | [View](./Zomato%20dataset%20EDA/) |

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
